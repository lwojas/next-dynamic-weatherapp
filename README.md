# Weather app

This app shows the current weather in a pleasing single page UI.

## Libraries / API's

- ** React / Next.js** 
- **Typescript**
- **Axios**  
- **Accuweather api**

## Features

- Search forecast by location
- Uses a debounce function to avoid spamming api calls
- Dynamic background based on current weather
